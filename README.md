
# Project Title

OpenICT portfolio Levi Leuwol 


## Run Locally

Clone the project

```bash
  git clone https://github.com/Shambuwu/openict-portfolio.git
```

Go to the project directory

```bash
  cd openict-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
